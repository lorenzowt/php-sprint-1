📚 PHP Concepts Practiced

This sprint focuses on intermediate OOP concepts in PHP, including:

* Abstract classes
* Object inheritance
* Polymorphism
* Method overriding
* Constructors (__construct)
* Shared behaviors between classes
* Extensible object-oriented architecture
* Class specialization through subclasses

⸻

📌 Notes

* Each exercise is a standalone PHP script
* Use inheritance to avoid duplicated code
* Use subclasses to specialize behaviors
* Keep responsibilities separated (one class = one purpose)
* Design reusable and scalable object-oriented structures
* Use print_r() or var_dump() for debugging objects when needed

⸻

🚀 Goal of this sprint

By the end of this sprint you should be comfortable with:

* Creating class hierarchies in PHP
* Applying inheritance and polymorphism correctly
* Designing reusable object-oriented systems
* Specializing behaviors using subclasses
* Structuring scalable PHP applications using OOP principles
* Thinking in terms of objects, behaviors, and relationships
