# Sprint 1 – s01-02 PHP Arrays

This folder contains the second set of PHP exercises focused on working with arrays.

The goal of this sprint is to practice:

- Indexed arrays
- Associative arrays
- Array manipulation
- Array iteration
- Array functions
- Functional programming helpers in PHP

This sprint introduces both basic and advanced array operations using native PHP functions.

---

# ⚙️ Requirements

To run these exercises you need:

- PHP installed (or XAMPP)
- VS Code (recommended)
- Terminal / command line access

### Run a file:

```bash
php filename.php
```

Example:

```bash
php level-1/ex01_basic_array.php
```

# 📚 PHP Functions Practiced

This sprint focuses heavily on native PHP array functions, including:

- `array_push()`
- `count()`
- `unset()`
- `array_values()`
- `array_map()`
- `array_filter()`
- `array_reduce()`

---

# 📌 Notes

- Each exercise is a standalone PHP script
- Prefer reusable functions when possible
- Keep array structures readable
- Use `print_r()` or `var_dump()` for debugging arrays

---

# 🚀 Goal of this sprint

By the end of this sprint you should be comfortable with:

- Creating and modifying arrays
- Working with associative arrays
- Iterating through arrays
- Using higher-order array functions
- Solving problems with array transformations
