# Sprint 1 – s01-04 PHP OOP 1
This folder contains the first set of PHP exercises focused on **Object-Oriented Programming (OOP)**.
The goal of this sprint is to practice:

- Classes and objects
- Constructors
- Encapsulation of data
- Inheritance
- Method overriding
- Basic OOP design principles

This sprint introduces fundamental OOP concepts using practical exercises and real-world inspired models.

📚 PHP Concepts Practiced

This sprint focuses on core OOP concepts in PHP, including:

* Class creation
* Object instantiation
* Constructors (__construct)
* Public methods and attributes
* Inheritance (extends)
* Method specialization in subclasses
* Working with object collections (arrays of objects)

⸻

📌 Notes

* Each exercise is a standalone PHP script
* Use classes to model real-world entities
* Keep responsibilities separated (one class = one purpose)
* Use arrays to manage groups of objects (e.g., movies, dice, shapes)
* Use print_r() or var_dump() for debugging objects when needed

⸻

🚀 Goal of this sprint

By the end of this sprint you should be comfortable with:

* Creating and using classes in PHP
* Building relationships between classes using inheritance
* Designing simple object-oriented systems
* Managing collections of objects
* Structuring PHP applications using OOP principles
