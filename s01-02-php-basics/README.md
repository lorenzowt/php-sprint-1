# Sprint 1 – s01-01 PHP Basics
This folder contains the first PHP exercises focused on programming fundamentals.
The goal of this sprint is to practice:
- Variables and data types
- Conditionals
- Loops
- Functions
- Constants
- Basic arithmetic operations
⚠️ Arrays are only introduced in Level 3.

- VS Code (recommended)
- Terminal / command line access
